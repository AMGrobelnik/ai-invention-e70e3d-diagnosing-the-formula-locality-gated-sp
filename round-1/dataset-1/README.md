## Layman Summary

This dataset pairs natural-language sentences with their logical formulas from FOLIO, ProofWriter, and MALLS, filtered for compositional complexity, to support experiments on AI translation of language into logic.

## Full Summary

This artifact provides a unified NL-FOL (Natural Language to First-Order Logic) dataset for the locality pilot experiment, assembled from three complementary sources:

**Primary: FOLIO (tasksource/folio, Yale-LILY/FOLIO, EMNLP 2024)**
- 1,892 examples: 1,862 pilot + 30 alignment-validation (stratified: 12 depth 2-3, 12 depth 4-5, 6 depth 6+)
- Gold human-annotated NL-FOL pairs extracted by splitting multi-premise FOLIO stories into individual (NL sentence, FOL formula) pairs
- All rows have non-empty gold FOL; depth-filtered to compositional_depth >= 2
- Fields: input (NL sentence), output (FOL formula), metadata_id, metadata_compositional_depth, metadata_fold (pilot/alignment_validation), metadata_story_id, metadata_label (True/False)

**Secondary: MALLS (yuan-yang/MALLS-v0, LogicLLaMA paper, ICLR 2024)**
- 993 examples tagged malls_supplement
- GPT-4 generated NL-FOL pairs, depth-filtered to compositional_depth >= 2 (range 2-7+)
- Diverse real-world statements with rich logical structure

**Supplementary: ProofWriter RelNeg-OWA (tasksource/proofwriter, Allen AI)**
- 272 examples tagged proofwriter_depth_ablation
- Relational NL sentences WITHOUT gold FOL (output field is empty string)
- Atomic sentences useful for depth-ablation analysis; compositional_depth sourced from QDep field
